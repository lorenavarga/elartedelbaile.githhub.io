¿COMO HACER PAGINAS CON HTML?

EN ESTA PAGINA,APRENDEREMOS COMO REALIZAR PAGINAS WEB CON HTML-APRENDIZ LORENA VARGAS


HTML significa lenguaje de marcas de hipertexto. No es un lenguaje de programacion per se,sino un lenguaje de marcado que se utiliza para crear la estructura de las paginas web.Define el significado y la estructura del contenido web.

  1. ESTRUCTURA BASICA DE UN DOCUMENTO HTML

    2. todo documento HTML comienza una declaracion de tipo de documento,seguido por elementos que define la cabeza (head) y el cuerpo(body) de la pagina.

  2. TITULOS Y VIÑETAS

    3. cuando pones un texto en una etiqueta (h1),el texto estara en negrilla,y la dimension de la letra sera igual al nuemro del encabezado.Los titulos pueden tener dimensiones entre 1 y 6.Las viñetas en HTML ofrece dos tipos principales de listas:

    • listas desordenadas (con vieñtas)
    • se utiliza con UL
    1. listas ordenadas (numeradas)
    2. se utiliza con OL

  3. ATRIBUTOS

    4. los elementos HTML pueden tener atributos que proporcionan inormacion adicional sobre el elemento. Por ejemplo, el elemento (img)tiene atributos como srs (que indica la fuente de la imagen) y alt ( que proporciona texto alternativo):

  4. BORDES

    5. para poder indicar un borde determinado con un color y un grosor concreto para un div, dentro del codigo CSS,utilizamos la propiedad borde,que utiliza el siguiente formato:
    borde: 2px solid grey;

  5. ELEMENTOS Y ETIQUETAS

    6. HTML utiliza "etiquetas" para marcar el inicio y el fin de los elementos.Estas etiquetas generalmente vienen en pares, por ejemplo:

    ETIQUETA DESCRIPCION
    head texto no visible(metadatos, codigos, palabras claves)
    title define el titulo de la pagina del navegador
    body encierra todo el contenido visible de la pagina
    ul, ol, li UL(viñetas), OL(numeros),y LI para los items,
    div y span contenedor para agrupar contenido de bloque y de linea,utilizado CSS.

  6. FORMULARIOS

    7. los formularios en HTML son elementos esenciales para recopilar informacion de los usuarios en una pagina web. Permiten enviar datos a un servidor para su procesamiento.






  7. ENLACES

    8. los enlaces (o hipervinculos) en HTML son elementos fundamentales que permiten navegar entre paginas web y recursos. Se crean con la etiqueta (a), con href se coloca el link,y con target="_blank"se abre el link en otra pagina.
